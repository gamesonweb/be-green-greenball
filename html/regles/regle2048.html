Règles du jeu 2048

Règles du jeu

Le jeu 2048 se joue sur une grille de 4x4 cases, sur laquelle sont placées des tuiles portant des nombres. Vous pouvez déplacer les tuiles dans quatre directions (haut, bas, gauche, droite). À chaque fois que vous déplacez une tuile, une nouvelle tuile apparaît aléatoirement sur la grille, avec une valeur de 2 ou 4.

Le but du jeu est de combiner les tuiles de même valeur pour créer une tuile avec la valeur 2048. Lorsque deux tuiles de même valeur se touchent, elles se combinent en une seule tuile dont la valeur est égale à la somme des deux tuiles d'origine. Si une tuile de valeur 2048 est créée, vous avez gagné la partie.

Le jeu se termine si toutes les cases de la grille sont remplies et que vous ne pouvez plus faire de mouvement. Dans ce cas, votre score final est le total des valeurs de toutes les tuiles présentes sur la grille.

À chaque fois que vous fusionnez deux tuiles, vous gagnez des points. Le nombre de points gagnés est 100 par tuiles fusionner.

Comment jouer

Pour jouer, utilisez les flèches du clavier pour bouger les tuiles dans la grille.

Choisir la difficulté

Le jeu 2048 est disponible en plusieurs niveaux de difficulté, qui modifient le temps avant la fin du chronomètre. Si le 2048 n'est pas atteint avant la fin alors vous avez perdu
